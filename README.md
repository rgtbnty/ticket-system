# ticket-system
A simple ticket management system built as a portfolio project.

## Overview
This project is a web-based ticket management system with authentication and CRUD features.

## Tech Stack
- Backend: Node.js
- Database: PostgreSQL
- Auth: JWT
- Infra: Docker

## API Testing
Frequently used API endpoints are automated using shell scripts.
```bash
source scripts/login.sh
./scripts/create_ticket.sh
```
